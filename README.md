# Superstore Profit Analysis

Tableau dashboard analyzing sales and profit performance 
across categories, regions, segments, and time.

## Live Dashboard
[View on Tableau Public](https://public.tableau.com/views/SuperstoreProfitAnalysisSpring2026_17737055253590/SalesAnalysis)

## Dataset
- Source: Tableau Sample Superstore Dataset (Professor shared the dataset)
- Fields: Orders, Returns, People — Category, Sub-Category, 
  Segment, Ship Mode, Region, State, Sales, Profit, Order Date

## Visualizations
- **Bar Chart** — Sales by Category and Sub-Category
- **Stacked Bar Chart** — Sales by Segment and Ship Mode
- **Filled Map** — Sales by State
- **Scatter Plot** — Profit vs Sales with linear regression 
  (p-value < 0.0001)
- **TreeMap** — Profit by Segment and Ship Mode
- **Line Chart** — Profit trend over time (2023-2026)
- **Box & Whiskers Plot** — Sales distribution by Region 
  and Sub-Category

## Key Insights
- Technology and Furniture lead in total sales
- Standard Class is the dominant shipping method across 
  all segments
- Strong positive correlation between Sales and Profit
- West region shows the highest sales variability

## Context
This project was developed as part of Data Science coursework 
to learn Tableau fundamentals including maps, statistical 
charts, and dashboard design.
